# AfCFTA Digital Trade Portal

A comprehensive digital platform facilitating trade across 54 African nations under the African Continental Free Trade Area (AfCFTA).

## Live Demo
**[View Live Application](https://afcfta-digital-trading.vercel.app/)**

## Features
- Government-backed identity verification (NIN, CAC, BVN)
- Real-time trade analytics across 54 African countries
- AI-powered trade recommendations
- Secure payment infrastructure (PAPSS integration)
- Comprehensive compliance center
- Live currency exchange rates (TradingView integration)

## Tech Stack
- **Frontend**: React 18 + TypeScript
- **Routing**: React Router v6
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Build Tool**: Vite
- **Deployment**: Vercel

## Installation

\`\`\`bash
# Clone the repository
git clone https://github.com/s8nclone/afcfta-digital-trading.git

# Navigate to project directory
cd afcfta-digital-trading

# Install dependencies
npm install

# Run development server
npm run dev
\`\`\`

No environment variables required for basic functionality.
